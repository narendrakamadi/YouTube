# Youtube

Developed a YouTube-inspired video streaming application using React and Redux Toolkit, following a scalable feature-based architecture. Implemented core features such as video listing, watch page, search functionality, and channel views, with centralized API management and clean separation of concerns to ensure maintainability and extensibility.

## Features

- Created skeleton of website
- Routing
- Header, sidebar, video feed and watch page layout
- Header UI Completed

## Packages Used

- npm install react-router-dom - Routing
